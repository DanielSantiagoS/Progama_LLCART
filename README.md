Código feito por Matheus Lopes e Daniel Sena Santiago. Este código tem como objetivo servir a churrascaria LLcart.
O sistema tem como finalidade cadastrar pedidos em comandas, vincular as comandas às mesas, cadastrar clientes e listar pedidos para a cozinha e para os funcionários responsáveis pelos pedidos.
